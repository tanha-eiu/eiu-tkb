Nạp file từ Phòng Đào tạo

Hệ thống chỉ trích xuất các lớp thuộc mã môn PHYS. File .docx cho độ chính xác cao nhất. Với file .pdf, hệ thống tự động chuyển đổi định dạng trước khi xử lý; độ chính xác tương đương nhưng không tuyệt đối, nên kiểm tra dữ liệu trước khi lưu.
